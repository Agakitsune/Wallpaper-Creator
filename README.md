# Wallpaper-Creator
A fast and low memory consuming application to create and manage wallpaper using X11

# Compability:

- Supports X11 Window (tested on i3)

- Supported Video Format:
    - mp4
    - asf
    - avi
    - drc
    - flv
    - M2TS
    - m4v
    - mkv
    - mpg
    - MTS
    - mxf
    - TS
    - vob
    - webm
    - wmv

- Supported but has problem:
    - ogv

- Not Supported (yet):
    - 3g2
    - 3gp
    - amv
    - rm
    - roq
    - yuv
